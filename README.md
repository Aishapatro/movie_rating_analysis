# Advanced Visualization in Movie Ratings

This project focuses on advanced data visualization techniques applied to movie rating datasets to uncover patterns, trends, and insights about audience and critic preferences.

## Overview

The project analyzes movie ratings using powerful visualization libraries to understand relationships between variables such as budget, genre, audience rating, critic rating, and year of release.

## Features

* Data cleaning and preprocessing of movie datasets
* Exploratory Data Analysis (EDA)
* Advanced visualizations including:

  * KDE plots (density plots)
  * Heatmaps
  * Pairplots
  * Jointplots
  * Correlation analysis
* Comparative analysis of audience vs critic ratings
* Insights into how budget and genre influence ratings

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

## Key Insights

* Relationship between budget and ratings
* Differences between critic and audience scores
* Genre-based rating trends
* Distribution of movie ratings

## Goal

The goal of this project is to demonstrate advanced data visualization skills and extract meaningful insights from movie rating data.

## Future Improvements

* Interactive dashboards using Plotly or Tableau
